# amazonLLD
set of standard LLD questions asked in Amazon LLD interviews

Learnings:
1. All standard LLD questions - 1. file filter, file downloader, booking system, chess, inventory management (vending machine), ATM Machine,
quick commerce, parking lot, amazon locker, rate limiter, LRU cache (support for multi-threading), elevator system.

2. ExecutorService, ScheduledExecutorService, Executors, Runnable, Thread.
3. Functional interfaces, lambda expression -> Consumer<T>, T Supplier, boolean Predicate<T> and Function<T,R>
4. Concurrent datastructure and its usage like LinkedBlockingQueue and ConcurrentHashMap, with function like put, take, compute, removeIf
5. Stream API -> intermediate function like filter(takes predicate) and map(takes function or supplier) which returns stream and terminal function like forEach, collect, count
6. File handling and Path -> usage of RandomAccessFile.
7. HttpURLConnection
